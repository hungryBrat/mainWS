## Course structure (handout)

11 modules: Intro → Mathematical Modelling → Components → RH stability → Time-domain → Root locus → Freq domain I (polar/Nyquist) → Freq domain II (Bode/lead-lag) → Model order reduction → Discrete-time → State space.
